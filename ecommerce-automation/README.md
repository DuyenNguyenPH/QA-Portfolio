# 🛒 E-commerce Website – Selenium Automation

## 🔍 Overview
Automation test scripts for a sample e-commerce website, written in Java + Selenium.

### 📌 Features tested:
- Login
- Product search
- Add to cart
- Checkout form validation

## ⚙️ Tools & Tech
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model

## 📂 Folder Structure
src/test/java
│
├── pages/       → Page Object classes
├── tests/       → Test cases
└── utils/       → Config & helper classes

## 🚀 How to run
1. Clone the repo
2. Import into IntelliJ or Eclipse
3. Install dependencies with Maven
4. Run test classes in `tests/`
